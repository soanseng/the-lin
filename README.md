# 林宅血案互動體驗網站

> 四十年未解的政治謀殺

一場發生在1980年白色恐怖時期的滅門血案，一段被層層掩蓋的調查真相。

本網站取材自三份政府公開調查報告，以沉浸式互動體驗呈現林宅血案的事件經過、四十多年的調查歷程，以及至今仍未解的諸多疑點。

## 資料來源

- 促進轉型正義委員會《林義雄宅血案調查報告》(民國109年)
- 高檢署《林宅血案、陳文成命案重啟調查偵查報告》
- 監察院 監112內正0004 糾正案文

## 開發

```bash
npm install     # 安裝依賴
npm run dev     # 啟動開發伺服器
npm run build   # 正式建置
```
